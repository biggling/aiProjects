### Top-Rated Features in Productivity/Focus Apps
1. **App blocking / distraction prevention** during sessions — #1 differentiator; highest impact on ratings
2. **Task-timer integration** — linking a Pomodoro to a specific task (Focus To-Do, TickTick)
3. **Minimalist, ad-free UI** — Goodtime built its reputation entirely on being ad-free; ads during focus sessions are a direct rating killer
4. **Gamification / streaks / virtual rewards** — Forest's tree mechanic, Habitica's RPG, Study Bunny's coins drive deep retention
5. **Widgets** — Sectograph's pie-chart clock widget is its core product; home screen widgets are a major differentiator
6. **Customizable intervals** — users want control beyond 25/5 defaults
7. **Cross-device sync + calendar integration** — TickTick and Todoist rated highly; Motion built entirely around this
8. **Analytics / productivity reports** — session history and streak visualization

## Kotlin + Jetpack Compose Updates

### Latest Compose Versions
- **Compose 1.8** (April 2025, BOM 2025.04.xx): Autofill stable, TextOverflow improvements, LookaheadScope stable, experimental APIs reduced from 172 → 70
- **Compose 1.10** (December 2025, BOM 2025.12.00) — **Most recent stable**: Scroll performance now matches Views, pausable composition in lazy prefetch, `retain` API for config changes, Material 3 1.4 (`SecureTextField`, `HorizontalCenteredHeroCarousel`)
- Source: Android Developers blog, Jetpack Compose release notes (2026-03-24)

### Best Libraries for QuickBlock
- **Room 3.0** (alpha as of March 2026, `androidx.room3`): KSP-only, coroutines-first, KMP support — migration path via Room 2.7.0+ with SQLiteDriver APIs
- **WorkManager 2.9.x** — correct tool for deferrable background work (notification reminders, streak calculations)
- **Hilt 2.51.1+** — recommended DI; compile-time graph validation, deep Jetpack integration
- **Standard NotificationCompat** + `POST_NOTIFICATIONS` runtime permission (required API 33+)

### Android API Restrictions — Impact on QuickBlock
| Area | Change | Impact |
|---|---|---|
| **Edge-to-edge** | Enforced by default on Android 15 | Must handle window insets in all screens |
| **Foreground Services** | Types required + permissions in manifest (API 34+); dataSync/mediaProcessing capped at 6h (API 35) | Use WorkManager instead of FGS where possible |
| **POST_NOTIFICATIONS** | Runtime permission since API 33 | Must request contextually; graceful fallback |
| **PACKAGE_USAGE_STATS** | Manual user grant via Settings (not runtime permission) | Use `ACTION_USAGE_ACCESS_SETTINGS` intent to guide user |
| **Full-screen intents** | Opt-in by default from Jan 2025 (only alarm/calling apps) | Cannot use full-screen notifications unless QuickBlock adds alarm feature |
| **Notification Cooldown** | Android 15: auto-reduces intensity for rapid bursts | Space out reminder notifications |

- **CRITICAL**: `PACKAGE_USAGE_STATS` is not a runtime permission — users must manually enable it. For QuickBlock's focus tracking (if added), UX must guide users to Settings.
- Android 14+ restricts real-time foreground app detection for privacy — use aggregate stats, not polling
- Source: Android Developers documentation, Android 15 release notes (2026-03-24)

## Thai Android Market

### Market Overview
- Android market share Thailand: **65.91%** (iOS: 33.68%) — Feb 2026, Statcounter
- Smartphone market grew **17.1% in 2024** to 16.9M units sold (IDC); Samsung + Chinese OEMs dominate Android
- Day 1 retention: **27%** (Asia average); Day 7: 8.1%
- Source: Statcounter, IDC Thailand report (2026-03-24)

### Thai App Popularity
- Top categories: Social (LINE, Facebook, TikTok), E-commerce (Shopee, Lazada), Finance (TrueMoney), Government (ThaID)
- Gaming IAP: **$162M in Q1 2025** — Thailand leads all SEA (Sensor Tower)
- Gaming market projected at **THB 13.4 billion (~$430M USD)** by 2025

### Thai-Language Productivity Gap — CONFIRMED
- **No dominant Thai-language-native productivity app** found in Thai Play Store top charts
- Top productivity apps in Thailand are all Western globals: Notion, Microsoft 365, Google Workspace
- Western AI tools "often struggle with Thai language nuances" — cited pain point
- Thai productivity startup density: effectively zero (Tracxn returned no substantive listings)
- **FLAG**: Thai-language UI for QuickBlock could improve Play Store ranking in Thailand region
- Source: Tracxn, Google Trends, Play Store analysis (2026-03-24)

### Thai Monetization Preferences
- **39% of Thai developers use advertising** (vs 27% globally) — heavy reliance on ads
- Only 9% use IAP (vs 10% global); 6% paid downloads (vs 3% global)
- Freemium with ads is dominant among local devs; users expect productivity apps to be free
- Willingness to pay exists but primarily for **entertainment** (gaming, streaming)
- Pricing sensitivity: **2–3x lower willingness to pay** than Western markets — use Google's regional pricing tiers
- Source: 42matters Thai Google Play publisher data, Sensor Tower SEA 2025 (2026-03-24)

### Alternative Distribution
- Sideloading (APKPure, APKMirror) widely used currently
- **MAJOR CHANGE**: Google rolling out "Accountability Layer" sideloading restrictions — **Thailand is one of 4 pilot countries** (with Brazil, Indonesia, Singapore), targeting **September 2026** — reinforces Play Store as dominant channel
- Source: Google Android developer announcement, Thailand MDES endorsement (2026-03-24)
