1. Copy the right CLAUDE.md from `1-stack-configs/` to your project root
2. Run `bash 2-memory-os/setup-memory-os.sh` from your project root
3. Copy `3-auto-mode-profiles/standard/settings.json` to `.claude/settings.json`
4. Copy `5-multi-agent-system/` scripts to your `scripts/` folder
