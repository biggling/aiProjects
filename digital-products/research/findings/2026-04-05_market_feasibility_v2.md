## 2. New Products Researched (April 2026)

### NEW #A — Cursor Rules Pack (.cursorrules / .mdc)
The parallel to CLAUDE.md configs, but for Cursor IDE.

**Market signals:**
- Cursor has $2B ARR and 2M+ users (faster-growing than Claude Code's user base by volume)
- Official Cursor Plugin Marketplace launched: packages .cursorrules/.mdc files, Skills, MCP definitions, Hooks
- `awesome-cursorrules` GitHub repo (PatrickJS) = free reference collection — same pattern as free CLAUDE.md GitHub repos
- cursorrules.org = free generator (same as aitmpl.com for Claude)
- "Cursor Rules Mega Pack" (DEV Community 2026) confirms market is forming
- Cursor Business ($40/seat) includes team-shared rules — team license opportunity identical to Claude Code

**Gaps confirmed:**
- No paid stack-specific .cursorrules packs on Gumroad found
- Cursor Marketplace is new — early mover window exists NOW
- Cross-tool bundle (CLAUDE.md + .cursorrules same ZIP) = unique TAM expansion, near-zero extra effort

**BiG fit:** HIGH — same Go/TS/Python/K8s expertise applies directly

| Factor | Score | Notes |
|---|---|---|
| Market demand | 9/10 | 2M Cursor users actively configuring IDEs |
| Competition | 8/10 | Free repos exist; no paid stack-specific packs confirmed |
| BiG's moat | 9/10 | Same engineering background applies |
| Build effort | Very Low | Adapt CLAUDE.md content to .cursorrules format |
| Price ceiling | $27–$49 | Same as CLAUDE.md pack |
| TAM expansion | Very High | 2M Cursor users on top of Claude Code market |

**Feasibility: GREEN — lowest-effort highest-TAM expansion available. Add to Phase 1.**
