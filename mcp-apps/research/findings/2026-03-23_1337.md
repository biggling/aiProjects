## IRS Per-Wallet Mandate — New Demand Amplifier

**IRS Rev. Proc. 2024-28** now requires per-wallet cost basis tracking from 2026:
- Each wallet tracked independently — no more pooled/universal cost basis
- Assets acquired before Jan 1, 2026 = user responsible for historical tracking
- Failure to report cost basis → IRS assumes cost basis = $0 → massive unjustified tax bill
- Form 1099-DA (2025 transactions) reports gross proceeds only for first year; **cost basis required starting 2026 tax year (forms out early 2027)**
- This creates TWO urgency waves: (1) 2025 return confusion NOW, (2) 2026 cost basis compliance deadline

**Why this amplifies the MCP opportunity:**
- Per-wallet tracking means users can no longer use simple aggregate tools
- They need exchange-by-exchange, wallet-by-wallet P&L + cost basis — exactly what a CEX+DeFi combined MCP provides
- No existing MCP addresses per-wallet tracking; even CoinStats MCP is aggregate only
