## Task 3 — MCP SDK & Technical Updates

### TypeScript SDK Status (as of 2026-03-23)
- **Latest stable: v1.27.1** (released Feb 24, 2026)
- **v1.28 / v1.29 / v2: Not released**
- v2 is in pre-alpha on main branch; stable v2 anticipated Q1 2026 (may slip to Q2)
- v1.x will receive bug fixes/security patches for 6 months post-v2 GA
- npm: https://www.npmjs.com/package/@modelcontextprotocol/sdk
- GitHub releases: https://github.com/modelcontextprotocol/typescript-sdk/releases

### v2 Preview Features (from ts.sdk.modelcontextprotocol.io/v2/)
- Runs on Node.js, Bun, Deno
- Ships: MCP server libraries (tools/resources/prompts, Streamable HTTP, stdio, auth helpers)
- Ships: MCP client libraries (transports, high-level helpers, OAuth helpers)
- Optional middleware for Express, Hono, Node.js HTTP

### Spec Status
- **Current stable spec: 2025-11-25** — unchanged; no new spec draft or RFC published in March 2026
- Next spec release: tentatively June 2026 (requires finalizing SEPs in Q1 2026)
- SEP-1686 (Tasks feature) refinements ongoing based on production feedback

### 2026 Roadmap Highlights (confirmed from official blog)
- Transport scalability: stateless Streamable HTTP sessions for horizontal scaling
- `.well-known` metadata format for server discovery without live connections
- Tasks primitive retry semantics and expiry policies
- Enterprise: audit trails, SSO, gateway behavior, config portability
- No new official boilerplate or starter templates mentioned
- Source: http://blog.modelcontextprotocol.io/posts/2026-mcp-roadmap/

### Notable Platform Integration (new this week)
- **Azure MCP in Visual Studio 2026**: now GA out-of-the-box — signals Microsoft's commitment to MCP as the standard
- **Google Cloud MCP**: fully managed remote servers, auto-enabled by service, IAM-governed — signals enterprise MCP maturation

### IRS 1099-DA Enforcement — Demand Signal Active
- Starting tax year 2025 (reported Feb 2026): brokers must report **both gross proceeds AND basis** for digital assets on Form 1099-DA
- February 17, 2026: brokers sent taxpayers 1099-DA copies — this is a live, real deadline
- IRS penalty waiver only for "good faith" filers in first year — confusion and complexity driving demand for AI assistance
- DeFi specifically flagged as high confusion/audit risk: "mismatch between how crypto transactions function and how the tax system assumes financial activity occurs"
- Tax Notes article (Mar 11, 2026): specifically covers 1099-DA implications for DeFi users
- Sources:
