## Task 2: Emerging MCP Use Cases — AI Agents & Automation

### Agentic Workflow Drivers

**MCP SDK downloads crossed 97M/month** (Python + TS combined) — confirmed by multiple sources including Dynatrace, Nevermined. Every major AI provider has adopted: Anthropic, OpenAI, Google, Microsoft, Amazon.

**2026 MCP Roadmap** (official): Moving in 4 directions:
1. Transport evolution (stateless sessions for scalability)
2. **Agent-to-agent (A2A) communication** — not just host-to-tool; MCP becomes the inter-agent bus
3. Governance maturation — audit trails, access policies
4. Enterprise readiness — security reviews, regulatory compliance

**CrewAI + MCP** (confirmed active March 2026):
- CrewAI 2.1 supports MCP server URLs directly in agent config
- Financial analysis crews being built: stock analysis reports, CRM-driven client reports
- **Near Intents + CrewAI MCP**: bridges AI + blockchain for "truly autonomous financial agents" — first autonomous DeFi agent infrastructure with MCP
- Token Metrics + CrewAI + Composio integration: live and functional

**DeFi Trading Agent MCP** (edkdev/defi-trading-mcp — already in Known Facts):
- Confirmed: DeFAI (DeFi + AI) is the emerging category — AI agents managing lending/borrowing/liquidity/rebalancing autonomously
- CoinGecko lists 550+ AI agent crypto projects, $4.34B combined market cap as of March 2026
- Key point: these are DeFi-only agents; **no MCP-based agent handles CEX portfolio P&L or tax**

**Enterprise Use Cases (Finance/Compliance)**:
- MCP acts as "backbone of financial institutions' risk and compliance automation" — per MintMCP and FinOps Foundation
- Common patterns: customer support with secure account context, compliance/recon queries over governed data, fraud ops workflows, portfolio/risk analysis against data warehouses
- **65% of financial institutions cite data security as primary AI barrier** — security/governance is the enterprise unlock
- ServiceNow GA'd MCP + A2A support (Zurich Patch 4) — enterprise workflows now natively MCP-aware
- Centralized MCP gateways provide unified auth + audit trails = regulated environment requirement

**DECISION TRIGGER:** Enterprise compliance MCP for crypto (sanctions screening, regulatory status, audit trail) is explicitly unaddressed by every current crypto MCP. Institutional buyers are forming. This is a longer-term ($149+/mo) tier opportunity.

## Task 3: Competitor Deep Dives — Gloria AI & ZARQ

### Gloria AI (Crypto Briefing MCP) — Full Feature List

**GitHub:** https://github.com/cryptobriefing/gloria-mcp

**Free tier tools (no payment required):**
- `get_latest_news` — latest curated crypto headlines with sentiment and categories
- `get_news_recap` — AI-generated hourly summary per category (crypto/macro)
- `search_news` — keyword-based news search
- `get_categories` — lists 18+ news categories
- `get_news_item` — retrieve specific news item by ID

**Paid tier tools (x402 USDC on Base micropayments):**
- `get_enriched_news` — full context, entity analysis, token mentions
- `get_ticker_summary` — 24h AI summary for any ticker/topic

**Infrastructure:** Gloria AI News Terminal — scans thousands of sources, filters for relevance, pushes structured updates. Integrates via API, WebSocket, Virtuals ACP, Coinbase x402. Clients: traders, content creators, AI agents, media firms.

**Authentication:** API token required even for free tier.

**Partnership:** EvalEngine integrated Gloria AI to power "ChromBot" real-time crypto data.

**What Gloria AI does NOT do:**
- No portfolio tracking of any kind
- No cost basis or P&L calculation
- No CEX account integration
- No tax reporting
- No Thai exchange coverage
- Crypto news + sentiment ONLY — purely informational

**Why 6,500 weekly PulseMCP visitors:** It's the only high-quality, maintained, multi-source crypto news aggregator as an MCP. No real competition for what it does. Its moat is editorial/curation quality + speed.
