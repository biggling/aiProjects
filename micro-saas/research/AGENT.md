# micro-saas — Research Agent

## Context
Building a focused Micro-SaaS solving one boring problem deeply for a specific audience.
Target: $5-20/month subscription, 10 paying customers before adding features.
Top idea: Shopee seller analytics dashboard for Thai market.

## Research Tasks

### 1. Thai SMB & Seller Pain Points
Search: "Shopee Thailand seller problems 2026", "Lazada seller analytics Thailand", "Thai e-commerce seller tools"
- What do Thai Shopee/Lazada sellers complain about most in Facebook groups and forums?
- Is there a paid analytics tool for Shopee TH sellers already? What does it cost?
- What metrics do sellers most want to track (GMV, profit, inventory, reviews)?

### 2. Indie SaaS Validation Research
Search: "micro saas $10/month success 2026", "indie hacker saas validation", "niche B2B SaaS revenue 2026"

- What validation methods work (landing page + waitlist, 5 customer calls, etc.)?
- How do successful micro-SaaS founders find first customers?

### 3. Shopee API Research
Search: "Shopee open API 2026", "Shopee seller center API", "Shopee partner API documentation"
- Does Shopee have a seller analytics API accessible to third parties?
- What data can be pulled via Shopee's Open Platform API?
- Any existing libraries for Shopee API in Python or Go?

### 4. Competitor Landscape
Search: "Shopee analytics tool", "Lazada seller dashboard tool", "Thai e-commerce analytics SaaS"
- Who are the existing competitors in Thai e-commerce analytics?
- What do they charge and what do they include?
- What's their weakest point (feature gap, UX, price)?

### 5. Alternative Micro-SaaS Ideas
Search: "Thai freelancer invoice tool", "Bangkok small business software 2026", "developer monitoring tool micro saas"
- Are there other underserved Thai SMB software niches?
- Any low-code invoice/accounting tools missing Thai market support?
