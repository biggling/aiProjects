## Context
Building a focused Micro-SaaS solving one boring problem deeply for a specific audience.
Target: $5-20/month subscription, 10 paying customers before adding features.
Target market: **global** — English-speaking, any timezone.
Top idea: e-commerce seller analytics (Shopee/multi-platform). Secondary: freelancer tools, developer utilities, SMB automation.

## Research Tasks

### 1. Opportunity Analysis — Idea Scoring
Search: "micro saas opportunity 2026", "niche SaaS underserved market", "B2B software gap indie hacker"
- Which global micro-SaaS ideas have a clear pain point with no good solution under $20/month?
- Rate each idea: pain intensity / willingness to pay / competition / API availability / build effort
- E-commerce analytics vs. freelancer invoicing vs. developer utilities — which scores highest?
- Any new pain points emerging from global indie/freelance/e-commerce communities in 2026?
- What "boring problems" do people pay $5-20/month to solve that haven't been well-addressed yet?
- Check IndieHackers, Reddit, HN "Ask HN: what tools do you wish existed"

### 2. Market Size & Growth
Search: "global freelancer market size 2026", "e-commerce seller tools market 2026", "micro SaaS TAM"
