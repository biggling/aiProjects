## Shop Architecture

### How Many Shops?

**Recommendation: 2 shops, launched sequentially**

| Shop | Name Theme | Primary Persona | Focus | Launch |
|------|-----------|-----------------|-------|--------|
| **Shop A** | `[Name]Gifted` or `MemorableMomentsShop` | Gen X Women + gift-buyers | Personalized milestone gifts, Mother's Day, Father's Day, retirement, anniversaries | **Week 1 — NOW** |
| **Shop B** | `[Name]Studio` or `AestheticLabStore` | Millennials + Gen Z | Aesthetic/identity designs: Romantic Goth, Literary Girl, Circus, Chateaucore, Star Maps | **Month 2 — Week 5** |

**Why 2 shops, not 1:**
- Etsy's algorithm rewards niche consistency. A shop that sells both "Grandma Gift" mugs and "Romantic Goth" sweatshirts confuses the relevance engine.
- Each shop builds its own review velocity and shop-quality score independently.
- The automated pipeline can publish to both simultaneously via separate API keys.

**Why NOT 3+ shops yet:**
- Under 5 hrs/week, managing 3 shops with reviews/customer service is too much overhead.
- Split into a 3rd shop (e.g., seasonal/holiday) only after Shop A reaches 50+ reviews (Month 4+).

## Path to 100 Listings/Week (Pipeline Ramp)

### Why 100/week is the target
- Etsy rewards continuous listing activity (314% higher traffic for weekly-upload shops)
- At 100 new listings/week, catalog doubles every 2–3 weeks initially
- This is only viable with the automated pipeline — impossible manually

### Ramp Schedule

| Week | New Listings/Week | Cumulative Total | What Unlocks This |
|------|-------------------|-----------------|-------------------|
| W1 | 20 | 20 | Manual setup; Mother's Day urgency |
| W2 | 20 | 40 | Pipeline Phase 1-2 running |
| W3 | 20 | 60 | Father's Day batch |
| W4 | 25 | 85 | Pride Month + crossbody bags |
| W5 | 50 | 135 | Shop B launches; pipeline dual-shop |
| W6 | 40 | 175 | Post-Mother's Day pivot batch |
| W7 | 40 | 215 | Automation stable on both shops |
| W8 | 40 | 255 | Graduation + summer batch |
| W9 | 50 | 305 | Scaled pipeline |
| W10 | 50 | 355 | Q3 preview content |
| W11 | 50 | 405 | Optimize + refill gaps |
| W12 | **100** | **505** | Full pipeline at capacity |

### Pipeline Configuration for 100/week

The existing pipeline (Phases 0–7) needs these settings to reach 100/week:

**Celery Beat schedule adjustments:**
```
trend_scraper:      Daily (niche discovery)
prompt_generator:   2× daily (morning + evening)
image_generator:    2× daily — 7–8 images per run = 14–16/day = 100/week
copy_generator:     After each image batch
etsy_uploader:      After copy, throttled to 15 listings/day max (avoid spam flag)
```

**Rate limiting rule:** Etsy flags rapid bulk uploads as spam. Cap at **15 new listings/day** per shop. With 2 shops = 30/day = 210/week maximum safe rate. Target 50–100/week is well within safe zone.

**Image generation cost at 100/week:**
- DALL-E 3: ~$0.04/image × 100 = $4/week = $16/month
- Stability AI: ~$0.01/image × 100 = $1/week = $4/month
- Claude (copy generation): ~$0.50–$1.00/week at 100 listings
- **Total pipeline cost at 100/week: ~$20–25/month**

## Weekly Keyword & Trend Research Process

### The 3-Source System (30 min/week every Monday)

#### Source 1: Automated Pipeline Research Agent (15 min review)
The existing research agent (AGENT.md) runs daily and produces findings in `research/findings/`. Every Monday:
- [ ] Read latest findings file
- [ ] Extract top 3 new niche signals
- [ ] Add them to the pipeline's `SEED_KEYWORDS` list in `tools/shared/config.py`
- [ ] Flag any seasonal deadline changes

#### Source 2: Etsy Free Research (10 min)
Open Etsy search and run these free techniques:
1. **Autocomplete mining:** Type your niche keyword, note all autocomplete suggestions → these are real buyer searches
   - E.g., type "mom gift" → autocomplete shows "mom gift from daughter personalized", "mom gift birthday unique", etc.
2. **Etsy search result count check:** Count results for target keywords — aim for 2K–10K for new shop viability
3. **"Best Sellers" filter:** Sort by "Most Relevant" and note top listing titles for each niche

#### Source 3: Free Trend Tools (5 min)
| Tool | What to check | Frequency |
|------|--------------|-----------|
| **Google Trends** | Compare 2–3 competing keywords; check 12-month trend direction | Weekly |
| **Pinterest Trends** (trends.pinterest.com) | Visual trend spikes; POD-relevant aesthetics | Weekly |
| **Etsy Trend Report** (seller-handbook) | Etsy's official seasonal reports | Monthly |
| **r/EtsySellers** (Reddit) | Real seller reports of what's selling | Weekly scan |
| **TikTok Search** | Search "Etsy haul" + your niche | Weekly |

**Update `tools/shared/config.py`:**
