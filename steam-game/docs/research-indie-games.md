## Key Patterns
1. **Price sweet spot:** $4.99-$9.99 for solo dev
2. **Simple mechanic, deep strategy:** One core input with emergent complexity
3. **Roguelike elements:** Runs, randomization, progression — infinite replay
4. **Pixel/minimalist art:** Achievable by solo dev, nostalgic appeal
5. **"One more run" loop:** Sessions 15-45 minutes
6. **Godot gaining traction:** Brotato, Dome Keeper proved Godot viable for hits

## Why "Sigil Spin" Fits
- Slot machine mechanic is instantly understood (simple input)
- Sigil combos create deep strategy (emergent complexity)
- Roguelike structure gives infinite replayability
- Pixel art sigils are minimal art scope
- Proven formula: Luck Be a Landlord validated the roguelike-slots genre
- Differentiation: combat element adds excitement that pure slots lack
